# vibe-coding-assignment
AI-based assignment manager developed with vibe coding process
# AI 기반 대학생 과제 관리 웹앱

## 1. 프로젝트 개요

본 프로젝트는 바이브코딩을 활용하여 대학생의 과제 일정을 관리하는 웹앱을 개발하는 것을 목표로 한다. 사용자는 과제를 등록하고, 마감일을 확인하며, 완료 여부를 관리할 수 있다.

## 2. 개발 목적

본 프로젝트의 목적은 단순히 SW를 완성하는 것이 아니라, 요구사항 분석부터 설계, 구현, 테스트, 품질 관리에 이르는 소프트웨어 개발 프로세스 안에서 바이브코딩의 효과와 한계를 분석하는 것이다.

## 3. 사용 기술

- Python
- NiceGUI
- JSON 파일 저장 방식
- GitHub 형상 관리
- ChatGPT 기반 바이브코딩

## 4. 주요 기능

- 과제 추가
- 과제 목록 조회
- 과제 완료 처리
- 과제 삭제
- 마감일 관리
- 완료/미완료 필터링

## 5. 저장 방식

본 프로젝트는 별도의 데이터베이스를 사용하지 않고 `tasks.json` 파일을 사용하여 과제 정보를 저장한다.

## 6. 개발 프로세스

- 요구사항 분석
- 화면 및 데이터 구조 설계
- AI 기반 코드 생성
- 실행 및 오류 수정
- 테스트
- 품질 관리
- 결과 분석
