# Design

## 1. 설계 목적

본 문서는 AI 기반 대학생 과제 관리 웹앱의 화면 구조, 데이터 구조, 주요 기능 흐름을 정리하기 위해 작성되었다. 본 프로젝트는 바이브코딩을 활용하여 구현하되, 요구사항 분석 결과를 바탕으로 기능 범위를 명확히 설정하고 설계 내용을 먼저 정리한 뒤 구현을 진행한다.

## 2. 전체 시스템 구조

본 프로젝트는 Python과 NiceGUI를 사용하여 웹 기반 사용자 인터페이스를 구성한다. 사용자가 웹 화면에서 과제를 입력하면, 프로그램은 해당 데이터를 내부 리스트에 저장하고 `tasks.json` 파일에 기록한다. 프로그램이 다시 실행될 때는 `tasks.json` 파일을 읽어 기존 과제 목록을 불러온다.

전체 구조는 다음과 같다.

```text
사용자
  ↓
NiceGUI 웹 화면
  ↓
Python 로직
  ↓
tasks.json 파일
