### Prompt

NiceGUI로 만든 과제 관리 웹앱의 실행 화면을 캡처하려고 하는데, 어떤 화면을 screenshots 자료로 저장해야 하는지 알려줘. 또한 전체, 미완료, 완료 필터 버튼을 눌렀을 때 어떤 필터가 선택된 상태인지 잘 구분되지 않는데, 선택된 버튼만 진하게 보이도록 UI를 수정하는 방법도 알려줘.

### AI 응답 요약

- AI는 실행 화면 캡처 대상으로 빈 목록 화면, 과제 추가 화면, 완료 처리 화면, 전체 필터 화면, 미완료 필터 화면, 완료 필터 화면, 입력값 검증 화면을 제안하였다.
- 각 스크린샷이 어떤 기능을 보여주는지 구분할 수 있도록 파일명을 정리하는 방식을 제안하였다.
- 전체, 미완료, 완료 필터 버튼의 선택 상태가 명확하지 않은 문제를 해결하기 위해 선택된 버튼만 진한 색으로 표시하는 UI 개선 방안을 제안하였다.
- `current_filter` 값을 기준으로 필터 버튼 영역을 다시 렌더링하여 선택된 필터 버튼과 선택되지 않은 버튼의 스타일을 다르게 적용하는 방식을 안내하였다.
- NiceGUI 실행 중 포트 충돌이 발생할 경우 `ui.run(port=8090)`처럼 다른 포트를 지정하여 실행하는 방법을 안내하였다.

### 검토 결과

- 제안된 스크린샷 항목은 기능별 동작 확인 자료로 활용할 수 있다고 판단하였다.
- 빈 목록 화면, 과제 등록 화면, 완료 처리 화면을 먼저 캡처하였다.
- 필터 버튼의 선택 상태가 잘 드러나지 않는 문제가 있어, 선택된 필터 버튼만 진하게 표시되도록 UI를 수정하였다.
- 수정된 UI를 기준으로 전체, 미완료, 완료 필터 화면을 다시 캡처하기로 하였다.
- 이후 입력값 검증 화면과 삭제 기능 화면도 추가로 캡처하여 제출 자료에 반영하기로 하였다.
