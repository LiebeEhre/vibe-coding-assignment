- 완료 여부 변경 시 completed 값을 반전시키고 JSON 파일에 저장하는 구조가 제안되었다.
- 삭제 기능은 과제 id를 기준으로 해당 항목을 제거한 뒤 JSON 파일을 갱신하는 방식으로 제안되었다.

### 검토 결과
- AI가 제안한 방식은 요구사항의 FR-04, FR-05에 해당한다.
- 완료 처리와 삭제 기능은 과제 관리 앱의 핵심 기능이므로 우선 구현 대상으로 적절하다고 판단하였다.
- 기능 추가 후 기존 과제 추가 기능과 JSON 저장 기능이 유지되는지 테스트가 필요하다.
